Informacion de como es la estructura 

Mirenlo como un arbol 
Menu (Padre): Es el archivo que controlara todo el flujo del proyecto, el cual le estara pasano constantemente los parametros a los metodos de edo y numerica 
Controles Edo y Numerica: Estos se encargan de como segun dice el nombre de controlar cada tema por separado. Aqui encontraran 2 scripts Control_Edo  o Control Numerica estos seran los encargados de 
controlar todo lo que se haga en estos temas llamando a los metodos visuales implementados en el otro archivo de esta carpeta por tanto el otro archivo es el que contiene todos los metodos que tengan que ver con lo visual 

Luego  se encontraran con la carpeta Logica en esta carpeta estaran igual separados en 2 carpetas distintas las cosas de edo y numerica, aqui se guardaran todos los archivos que solamente tengan que ver con la logica 
del proyecto en si.

Pueden crear los archivos que deseen en estas carpetas siguiendo esa estructura para asi mantener el proyecto lo suficientemente organizado y limpio
